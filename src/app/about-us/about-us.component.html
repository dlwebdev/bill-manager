About BillManagementSystem

With BMS you have one place to track and manage all of your finances!

Application Details

BillManagementSystem was created for my fullsail WD3 class project.

BMS is a single page application(SPA), also known as a CRUD application which stands for create, read, update, delete.

It was created using the MEAN stack, which consist of Mongo DB, Express.js, Angular 2, and Node.js.
